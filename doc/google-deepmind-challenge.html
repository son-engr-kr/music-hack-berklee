Google DeepMind Challenge — Magenta RealTime 2

Prize: $2,000 cash for the winning team

Overview

Build and play your own live AI instruments with Magenta RealTime 2, an on-device music model provided as an AudioUnit and Python library. It supports MIDI control, audio/text prompt steering, <100ms response time, runs locally on a modern MacBook, and is open source.

Challenge

Create a live music instrument, plugin, app, experience, and/or performance that blends genres and creates never before heard sounds. Push past static "text2music" toward live, embodied, social, and interactive AI music.

What Teams Can Build

Anything using Magenta RealTime 2 as the creative engine:

  • Personalized instrument for live performance
  • Interactive/reactive music installation
  • EEG headset brainwave-controlled music
  • Real-time jam partner
  • Assistive tool for music therapy

Getting Started

Landing page, GitHub code, Hugging Face models, blog post, and announcement. During the hackathon: model access, codebase, examples, and plugins for AudioUnit, MaxMSP, PureData, and SuperCollider. Magenta team mentorship available.
