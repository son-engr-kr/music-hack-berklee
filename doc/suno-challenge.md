# Suno Challenge

**Prize:** $2,000 cash for the winning team

## Overview

Build new ways to play with music using the Suno API. Turn ideas, sounds, and moments into music through prompt-based generation and audio-based creation, remixing, and interaction.

## Challenge

Create a project that helps people make, transform, or experience music in a new way — a creative tool, interactive experience, remixing workflow, listening experience, or something entirely unexpected.

## What Teams Can Build

Anything using the Suno API as part of a music experience:
- Tools for creating songs from ideas
- Remixing audio
- Collaborating with others
- Generating adaptive soundtracks
- New ways to interact with music

## Getting Started

Participants who select the Suno challenge receive an API access link, credits, and documentation by email ahead of the event.
