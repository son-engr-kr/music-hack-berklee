AI Sound Canvas

사운드를 픽하고, 클론하고, 쌓아서 나만의 새로운 악기를 만든다

개요

AI Sound Canvas는 웹 기반의 인터랙티브 신디사이저로, Magenta (DDSP)를 엔진으로 사용한다. 사용자는 세 가지 입력 방식(A/B/C)으로 소스를 제공하고, 그 안에서 원하는 사운드를 픽하여 클론한 뒤, 여러 레이어를 블렌드해서 하나의 새로운 악기로 탄생시킨다.

모든 과정은 브라우저 위에서 동작하며, 최종 결과물은 웹 MIDI 키보드로 연주하거나 오디오로 익스포트할 수 있다.

시스템 흐름

┌──────────────────────────────────────┐ │ INPUT SOURCE │ │ │ │ [A] 오디오 파일 업로드 │ │ [B] 텍스트 프롬프트 │ │ [C] 허밍 / 마이크 입력 │ └──────────────┬───────────────────────┘ │ ▼ ┌──────────────────────────────────────┐ │ MAGENTA DDSP ENGINE │ │ → Timbre Transfer & Sound Analysis │ │ → 스펙트럴 특징 추출 & 복제 │ └──────────────┬───────────────────────┘ │ ▼ ┌──────────────────────────────────────┐ │ SOUND PICK & CLONE │ │ "이 신스 소리 골라줘" │ │ → 특정 구간의 음색 캡쳐 │ │ → 파라미터 맵핑 (pitch, filter, env) │ └──────────────┬───────────────────────┘ │ ▼ ┌──────────────────────────────────────┐ │ LAYER STACK │ │ │ │ ┌─ Layer 1: 클론된 신스 (source A) ─┐ │ ├─ Layer 2: AI 생성 악기톤 (prompt) ─┤ │ ├─ Layer 3: 내 허밍 (source C) ─┤ │ ├─ Layer 4: 또 다른 클론 ─┤ │ │ │ │ │ 각 레이어 컨트롤: │ │ │ Blend % | Delay | Pan | Filter │ │ │ Volume | On/Off | Stereo Spread │ │ └────────────────────────────────────┘ │ ▼ ┌──────────────────────────────────────┐ │ PLAYABLE INSTRUMENT │ │ 하나의 새로운 악기로 탄생 │ │ → Web MIDI Keyboard로 연주 │ │ → 오디오 Export (WAV) │ └──────────────────────────────────────┘

핵심 기능

🎯

Sound Picker

업로드한 노래 안에서 "이 소리"를 선택하면 DDSP가 그 음색을 분석 & 클론한다. 특정 구간의 신스 패드, 베이스, 리드 사운드를 캡쳐 가능.

🧪

AI Sound Generator

텍스트 프롬프트로 원하는 악기톤을 생성한다. "warm analog pad", "aggressive saw lead" 같은 설명 → DDSP가 실시간 생성.

🎤

허밍 & 마이크

입으로 내는 소리, 비트박스, 멜로디 허밍을 분석해서 피치와 음색을 추출, 신디사이저 파라미터에 매핑.

🧩

Layer Blender

각 레이어의 Blend %, Delay, Pan, Volume, Filter Cutoff, Stereo Spread를 실시간으로 조절. 단순 볼륨 믹스가 아니라 주파수 대역별 블렌드도 가능.

UI / UX 구성 (안)

메인 화면 — Layer Canvas

  • 상단: 입력 소스 선택 탭 (Upload / Prompt / Microphone)
  • 중앙: 각 레이어가 가로로 나열, 각 레이어마다 실시간 파형/스펙트로그램 시각화
  • 하단: 글로벌 컨트롤 (Master Volume, BPM, Key)

레이어 카드

  • 레이어 이름 / 소스 아이콘 (업로드 / AI / 마이크)
  • Sound Preview — 현재 레이어 단독 솔로 청취
  • 컨트롤 슬라이더:
    • Blend — 전체 믹스에서 차지하는 비중 (%)
    • Delay — 딜레이 타임 (ms), 피드백
    • Pan — 좌우 위치
    • Filter — Low/High/Band Pass + Cutoff + Resonance
    • Stereo Spread — 스테레오 이미지 폭
    • Volume — 레이어 게인
  • On/Off 토글 — 각 레이어를 켜고 끄면서 A/B 비교

Sound Picker 모달

  • 업로드한 곡의 파형 전체 표시
  • 구간 선택 → 분석된 스펙트럼 표시
  • "이 소리 클론" 버튼
  • 클론된 소리가 새 레이어로 추가됨

연주 화면

  • Web MIDI Keyboard (Qwerty 키보드 or MIDI 컨트롤러 연결)
  • 각 레이어가 동시에 연주됨
  • 연주 녹음 → Export WAV

데모 시나리오 (심사위원용)

  1. STEP 1 — 레퍼런스 곡 업로드
    🎵 80s 시티팝 한 곡을 업로드한다. 파형과 스펙트로그램이 표시된다.
  2. STEP 2 — Sound Picker로 신스 캡쳐
    🎯 곡의 인트로에서 "이 신스 패드 소리"를 선택 → DDSP가 음색을 분석하고 클론한다. Layer 1에 추가.
  3. STEP 3 — AI로 악기톤 생성
    🤖 "warm sub bass, round attack" 입력 → Magenta가 새로운 악기톤을 생성. Layer 2에 추가.
  4. STEP 4 — 허밍으로 멜로디 추가
    🎤 입으로 멜로디를 허밍 → 피치와 음색을 분석해서 Layer 3에 매핑.
  5. STEP 5 — 레이어 블렌드
    🧩 각 레이어의 Blend / Delay / Pan / Filter를 실시간 조절하며 최적의 믹스를 찾는다. Layer 1(클론 신스) 60% + Layer 2(AI 베이스) 30% + Layer 3(허밍) 10%.
  6. STEP 6 — 연주 & 익스포트
    🎹 웹 키보드로 만들어진 악기를 연주한다. 세 개의 레이어가 하나의 악기로 동시에 울린다. 결과물을 WAV로 Export.

차별성 (Why this wins)

  • Sound Picker: 단순히 "소리 생성"이 아니라, 기존 음악에서 원하는 사운드를 골라서 복제한다는 발상이 참신
  • Layer Blending: 단순 믹서가 아니라 딜레이, 필터, 스테레오 스프레드까지 각 레이어별로 조절 가능한 인터랙티브한 UX
  • 웹 네이티브: 설치 없이 브라우저에서 모든 동작 — 데모 및 피드백 사이클이 빠름
  • 세 가지 입력 방식: 누구나 자신에게 편한 방식으로 참여 가능 (파일 / 텍스트 / 마이크)
  • 악기로 탄생: 단순한 오디오 생성이 아니라 "연주할 수 있는 악기"가 결과물 — 참여형 데모에 강함
